KeyRhythm – Privacy Policy

Last updated: March 9, 2026

Overview

KeyRhythm is a rhythm game browser extension. Your privacy is important — this extension is designed to work entirely offline and does not collect, transmit, or share any personal data.

Data Storage

All data created while using KeyRhythm is stored locally on your device using browser storage APIs (IndexedDB and localStorage). This includes:

  • Songs, audio files, and level data you create or import
  • Playlists and custom backgrounds
  • High scores and gameplay statistics
  • Settings and preferences (keybinds, scroll speed, volume, themes)
  • Custom skins

None of this data is sent to any server or third party. It remains entirely on your device and under your control.

Data Collection

KeyRhythm does not collect any data. Specifically:

  • No personal information is collected
  • No analytics or telemetry is used
  • No cookies are set or read for tracking purposes
  • No user accounts or authentication are required
  • No data is transmitted to external servers

Permissions

The extension requests the following browser permissions:

  • Storage — to save your songs, settings, and scores locally.
  • Scripting & Tabs — to inject the Live mode overlay onto web pages when you explicitly enable it.
  • Offscreen — to play background music while the extension popup is closed.
  • Host permissions (optional) — requested only when you activate Live mode on a webpage. This allows the extension to capture audio from that page for real-time pitch detection. Audio is processed locally and is never recorded or transmitted.

Live Mode

Live mode lets you detect audio playing on a webpage and generate gameplay notes in real time. When you enable Live mode:

  • The extension requests permission to access the active tab's audio
  • All audio analysis (pitch detection) is performed locally in your browser
  • No audio data is recorded, stored, or sent anywhere
  • The overlay is removed when you disable Live mode or navigate away

Third-Party Services

KeyRhythm does not use any third-party services, APIs, or CDNs. All code and resources are bundled with the extension and run locally.

Data Sharing

KeyRhythm includes an export feature that lets you share levels via share codes or file export. This is entirely user-initiated — you choose when and how to share your data. The extension itself never shares data automatically.

Data Deletion

You can delete your data at any time by:

  • Deleting individual songs, scores, or playlists from within the extension
  • Resetting settings to defaults
  • Uninstalling the extension, which removes all associated local data

Changes to This Policy

If this privacy policy is updated, the changes will be reflected on this page with an updated date. The extension will continue to operate with the same commitment to user privacy.

Contact

If you have questions about this privacy policy, you can reach out by opening an issue on the project's GitHub repository.
